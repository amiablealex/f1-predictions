About

What this is

A small F1 prediction league for friends. Every Grand Prix weekend you submit a set of predictions before qualifying starts; closer guesses score more points.

Who made it

Built by amiablealex as a personal project.

The source code lives at github.com/amiablealex/f1-predictions. Bugs, ideas, or pace notes welcome.

How it works

Python and Flask on a Raspberry Pi for development; Postgres on Railway in production. Race data comes from the Jolpica F1 API, an open community-run successor to Ergast. Scoring runs once each session's official results are published, usually within an hour of the chequered flag.
